  • HTML TAG

    • HTML 태그들을 무자비하게 정리

    *꺾쇠는 . 으로 대체

    **대부분 작성 순서 및 중요도 순서대로 작성


    • .!DOCTYPE HTML / .HTML

      • HTML의 원활한 구동을 위해 꼭! 맨 위에 적고 시작

    • .head

      • 페이지의 가장 중요한 정보를 나타내는 부분.

    • .title

      • 페이지의 이름(?)을 바꿔줌. 검색에서 우선정보가 됨

    • .meta charset="UTF-8"

      • 페이지의 버전(형식). 이걸 안 적으면 한글이 안나올수도 있음. 현재는 대부분 UTF-8로 되니, 저대로 적자


    • .h1~6

      • 제목을 표현하는 태그. 각 숫자별로 크기가 다르며, 일반 p태그에 이미지 조정보다 강력한 영향이 있으니 중요할 때 꼭 활용할것

    • .p

      • 페이지의 단락을 결정하는 태그. 웬만하면 이걸로 쓴다

    • .br

      • 엔터

    • .ol, ul

      • 번호가 있는/없는 리스트 작성의 부모.

    • .li

      • ol, ul의 자식 두 태그안에 이것들을 적는 만큼 리스트작성이 된다.


    • .a href=""

      • 하이퍼링크 기능. 폴더에서 이미지 업로드 시, 꼭 경로를 정확히 할 것!

      ex) 폴더/이미지.확장자

      target="_blank" 를 붙인다면, 새 창에서 열기 /

    • .a id=""

      • a id="여기" 와 a href="#여기" 를 통해 해당페이지 바로가기 가능

      사실 뭐하는 놈인지 잘 모른다.

    • .img src="이미지 링크 or 사이트 폴더 내에 사진명"

      • 이미지 삽입. Width="" ->이미지 크기조정

      alt="" -> 이미지의 대체텍스트. 설명문이 되어 여러모로 검색에 용이해진다

    • .viedo src="이미지 링크 or 사이트 폴더 내에 사진명"

      • 동영상 삽입. 얘는 뒤에 /로 닫아줘야함. 꼭!

      Width="", heihgt="" ->동영상 크기조정

      controls -> 영상의 속도 등 조정가능한 옵션


    • .!--comment-->

      • 주석 이라는데...스텔스 기능인듯

    • .strong, u, em

      • 글의 굵기, 밑줄, 기울임
